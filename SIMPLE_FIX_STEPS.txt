================================================================================
                    SIMPLE FIX FOR POLICY CREATION ERROR
                        (NO CODING SKILLS NEEDED)
================================================================================

PROBLEM: When adding Motor/Health/Life policy, you get error.

CAUSE: Your Hostinger database is missing some columns.

FIX TIME: 5 minutes

================================================================================
                              STEP-BY-STEP FIX
================================================================================

STEP 1: COMMIT & PUSH YOUR CURRENT CHANGES
-------------------------------------------
1. Open GitHub Desktop
2. You should see changed files
3. Write in commit message: "Fix policy modal errors"
4. Click "Commit to main"
5. Click "Push origin" button at top
6. Wait for Hostinger to auto-deploy (1-2 minutes)


STEP 2: LOGIN TO HOSTINGER
-------------------------------------------
1. Go to: https://hostinger.com
2. Login with your credentials
3. You'll see hPanel (hosting control panel)


STEP 3: OPEN PHPMYADMIN
-------------------------------------------
1. In hPanel, look for "Databases" section
2. Click "phpMyAdmin" 
   OR
   Click "Manage" next to your database name
   Then click "Enter phpMyAdmin"


STEP 4: SELECT YOUR DATABASE
-------------------------------------------
1. On the LEFT side, you'll see database names
2. Click on your insurance database
   (probably something like: u123456_insurance)
3. You should see a list of tables


STEP 5: RUN THE FIX
-------------------------------------------
1. Click the "SQL" tab at the TOP
2. You'll see a big empty text box
3. Go to your project folder
4. Open file: "FIX_PRODUCTION_DATABASE.sql"
5. Press Ctrl+A (select all)
6. Press Ctrl+C (copy)
7. Go back to phpMyAdmin
8. Click in the SQL text box
9. Press Ctrl+V (paste)
10. Click "Go" button (bottom right)
11. Wait 5 seconds
12. You should see: "Database columns updated successfully!"


STEP 6: TEST YOUR WEBSITE
-------------------------------------------
1. Go to your website: v2insurance.softpromis.com
2. Hard refresh: Press Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
3. Try adding a Motor policy
4. Fill all required fields
5. Click "Add Policy"
6. IT SHOULD WORK NOW! ✅


================================================================================
                            IF SQL SCRIPT FAILS
================================================================================

If the big SQL script gives errors, use this simpler version instead:

IN PHPMYADMIN SQL TAB, PASTE THIS:

ALTER TABLE policies ADD COLUMN customer_age SMALLINT UNSIGNED NULL;
ALTER TABLE policies ADD COLUMN customer_gender VARCHAR(255) NULL;
ALTER TABLE policies ADD COLUMN sum_insured DECIMAL(12, 2) NULL;
ALTER TABLE policies ADD COLUMN sum_assured DECIMAL(12, 2) NULL;
ALTER TABLE policies ADD COLUMN policy_term VARCHAR(255) NULL;
ALTER TABLE policies ADD COLUMN premium_frequency VARCHAR(255) NULL;

Click "Go"

Note: You might get "Duplicate column" errors - that's OK! Ignore them.
